# 🌾 India Agricultural Crop Production Analysis (1997–2021)

## 📌 Project Overview

This project analyzes India's agricultural crop production data from 1997–2021 using Tableau and Flask. It provides interactive dashboards and stories to understand production trends, cultivated area, seasonal patterns, and state-wise crop performance.

---

## 🎯 Objectives

- Analyze crop production across India
- Compare state-wise agricultural production
- Study seasonal production trends
- Visualize cultivated area by crop
- Publish an interactive Tableau Story
- Integrate Tableau into a Flask website

---

## 📂 Dataset

- Source: India's Agricultural Crop Production Dataset
- Format: Excel (.xlsx)
- Years Covered: 1997–2021
