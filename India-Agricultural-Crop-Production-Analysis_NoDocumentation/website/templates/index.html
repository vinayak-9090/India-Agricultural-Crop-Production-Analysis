India's Agricultural Crop Production Analysis (1997–2021)

This project analyzes agricultural production across India using Tableau and Flask. It helps identify crop production trends, seasonal patterns, cultivated area, and leading agricultural states.

Objectives

  • Analyze crop production from 1997–2021.
  • Compare state-wise and crop-wise production.
  • Visualize trends using interactive dashboards.
  • Support data-driven agricultural insights.
